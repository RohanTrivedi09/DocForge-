Wire directly to `python-docx` header/footer with left+right tab stops:

```python
from docx.oxml.ns import qn
from docx.oxml import OxmlElement
from docx.shared import Pt
from docx.enum.text import WD_TAB_ALIGNMENT

def set_header(section, left_text, right_text):
    header = section.header
    para = header.paragraphs[0]
    para.clear()
    # Add tab stop at right margin
    pPr = para._p.get_or_add_pPr()
    tabs = OxmlElement('w:tabs')
    tab = OxmlElement('w:tab')
    tab.set(qn('w:val'), 'right')
    tab.set(qn('w:pos'), '9360')  # ~16.5cm right margin position
    tabs.append(tab)
    pPr.append(tabs)
    run1 = para.add_run(left_text)
    run1.font.size = Pt(10)
    para.add_run('\t')
    run2 = para.add_run(right_text)
    run2.font.size = Pt(10)
    # Add bottom border to header
    pBdr = OxmlElement('w:pBdr')
    bottom = OxmlElement('w:bottom')
    bottom.set(qn('w:val'), 'single')
    bottom.set(qn('w:sz'), '4')
    bottom.set(qn('w:space'), '1')
    bottom.set(qn('w:color'), '000000')
    pBdr.append(bottom)
    pPr.append(pBdr)
```

Same pattern for footer with left_text=lab_number, right_text=enrollment_no.

### Cell rendering rules (match PDF reference exactly):

**Markdown cells:**
- `# H1` → Word Heading 1, Times New Roman 16pt bold
- `## H2` → Heading 2, 14pt bold
- `### H3` → Heading 3, 12pt bold
- Body text → TNR 12pt
- Bold/italic preserved
- Bullet lists → Word list paragraphs

**Code cells:**
- Background shading `#F2F2F2` (light gray)
- Font: Courier New 10pt
- Syntax coloring via token parsing:
  - Keywords (`import`, `def`, `for`, `with`, `return`, `in`, `if`) → `#0000FF` (blue)
  - Strings → `#BA2121` (dark red)
  - Comments (`#...`) → `#3D7A00` (green)
  - Numbers → `#008000`
  - Default/identifiers → black
- Use Python's `tokenize` module to parse tokens, apply color per run in python-docx
- Label above each code cell: `In [N]:` in gray italic TNR 10pt (match notebook style)

**Text output cells:**
- Courier New 10pt, no shading, indented 0.5cm left
- Prefixed with nothing — just the raw output text

**Plot/image outputs (base64 PNG):**
- Decode and embed via `doc.add_picture()`, max width 12cm, centered
- Caption below: `Figure N` in gray italic 9pt, centered (auto-increment)

**DataFrame/table outputs:**
- Word table, auto-fit columns
- First row: bold, shading `#D9D9D9`
- Alternating rows: white / `#F7F7F7`
- Thin borders all cells

**Warning/stderr outputs:**
- Skip silently OR show in light yellow `#FFFBE6` box, Courier New 9pt gray text — add toggle "Show warnings" (default OFF)
